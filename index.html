Token personnel GitHub (ghp_...)
📖 Comment configurer ?
  1. Crée un compte sur github.com si tu n'en as pas
  2. Crée un nouveau dépôt public ou privé nommé chronos-data — initialise-le avec un README
  3. Va dans Paramètres → Developer settings → Fine-grained tokens → Generate new token
  4. Nom : Chronos — Expiration : selon ta préférence
  5. Dans Repository access → sélectionne Only select repositories → choisis chronos-data
  6. Dans Permissions → Repository permissions → Contents → mets Read and write
  7. Clique Generate token, copie le token (github_pat_...) et colle-le ci-dessus
  8. Entre ton dépôt au format username/chronos-data
⚠️ Le token n'est jamais exporté ni partagé — il reste uniquement sur cet appareil.
💡 Fine-grained token = accès limité à ce seul dépôt, plus sécurisé qu'un token classique.
